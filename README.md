## Quickstart

1. Fork this repo and clone your fork:

```bash
$ git clone https://github.com/<your-username>/tahaemporium
```

2. Client:

```bash
$ cd client
$ npm install
$ npm start
```

3. Server (create `.env` in `server` with your secrets):

```bash
$ cd server
$ npm install
$ npm start
```

## Notes

- Replace the BaseURLs in `client/src/api/BaseURLs.js` and each service `services/BaseURLs.js` with your deployed domains.
- Replace the logo assets in `client/src/shared/assets` with your TAHA EMPORIUM logo.
